# PRSK Design System Components

This directory documents the 20 custom PRSK components built on top of shadcn/ui.

## Forms & Inputs
1. `<RupeeInput>` - Indian currency formatting
2. `<GSTINInput>` - GSTIN mask & validation
3. `<PANInput>` - PAN mask & validation
4. `<PhoneInput>` - Indian mobile with WA badge
5. `<PincodeInput>` - Postal code lookup
6. `<IFSCInput>` - Bank IFSC lookup
7. `<DateInput>` - DD/MM/YYYY formatting

## Display & Data
8. `<MoneyDisplay>` - Currency formatting
9. `<TrustBadge>` - Verification status
10. `<ShowWorkingButton>` - Calculation breakdown
11. `<StatusPill>` - Semantic states
12. `<RoleBadge>` - User roles
13. `<ActivityItem>` - Feed items

## States
14. `<EmptyState>` - No data handling
15. `<ErrorState>` - Failure handling
16. `<LoadingSkeleton>` - Skeletons matching layouts

## Patterns
17. `<CommandPalette>` - Global search
18. `<BulkActionBar>` - Multi-select actions
19. `<FilterBar>` - List controls
20. `<UndoToast>` - Destructive action recovery
